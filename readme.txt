== Description ==
Server Status For Minecraft PC & PE is a WordPress Widget, show Minecraft PC & PE server data.

GitHub: https://github.com/MCNewsTools/MCServerStatus-WordPressWidget

You can display / hide these information:

 * Server name (MOTD)
 * Status (online/offline)
 * IP address / hostname
 * Port
 * Server platform
 * Server software
 * Game version
 * Website
 * DynMap
 * Plugins count
 * Plugins list
 * Player count (current / maximum)
 * Player list
 * Player skin avatar

and more features are planned.

Other functions:

 * Display web site link
 * DynMap
 * Set player avatar size
 * Auto update status

== Installation ==
1. Upload all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress

== Frequently Asked Questions ==
= Why can not I get server data? =
This method uses GameSpy4 protocol, and requires enabling query listener in your server.properties like this:

- enable-query=true

- query.port=25565
